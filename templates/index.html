What this run does

This is an email-only workflow now.

  • Zeus inventory is treated as the truth source.
  • All Zeus in-stock stock numbers are removed from the KeyTrak list.
  • What remains in KeyTrak is likely sold or stale inventory.
  • The result CSV is emailed to you.
  • Uploaded source files are deleted after processing.
  • All activity is written to logs/app.log.
• Zeus sample format: stock number in column 2.
• KeyTrak sample format: stock number in column 1, with first 2 rows skipped automatically.
